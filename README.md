# [Vancouver Jazz Exchange](https://vanjazz.gondek.ca/)

A meetup to discuss jazz and socialize.

**Build Requirements**
- `ruby` and Bundler
- `just` command runner (see [justfile](./justfile))

**Local Development**
1. `just install`
2. `just serve`

**Deployment**
1. `git commit` changes
2. `git push`
